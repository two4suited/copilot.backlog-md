## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Root cause:** The lint-orchestrator job in .github/workflows/ci.yml uses `cache: npm` on setup-node and runs `npm ci`, but the root package.json has no corresponding package-lock.json committed.

**Error:** `Dependencies lock file is not found in /home/runner/work/copilot.backlog-md. Supported file patterns: package-lock.json,npm-shrinkwrap.json,yarn.lock`

**Affected runs:** all CI runs since TASK-1.4 was merged

**Fix:** Generate and commit package-lock.json at repo root OR remove `cache: npm` from lint-orchestrator and use `npm install` instead of `npm ci`.
